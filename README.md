# DTA Client

Crystal Client for Ruby on Rails apps using [devise_token_auth](https://github.com/lynndylanhurley/devise_token_auth)

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     dta_client:
       github: HCLarsen/dta_client
   ```

2. Run `shards install`
